Data Science projects I have worked on


Project 1: Intelligent system for traffic signs detection

Road safety modeling is a valuable strategy for promoting safe mobility, through the development of this application I tried to build a model that detects and classify traffic signs in real time using tensorflow and openCV library. The application is deployed through a mobile app using flutter in order to alert the drivers and make them aware of the road environment.

Project 2: Marking attendance Web Application

Using facial recognition, I've built this application to mark attendance,time-in and time-out of employees. A lot of tools are involved in this, such us face_recognition, Dlib and also OpenCV library. I managed to process the images in real time and implement the model through a web application developed using Django to help manage the employees(Add new employee, train the model, view attendance report...)

Project 3: Credit Card Fraud Detection

Interactive web application using Streamlit and Flask, divided to three parts:EDA to visualize the dataset, analyse it , normalise and scale it, it the Training part using one or various models (Aggregator Model). Then the last part Inference to test and display if the the transaction is fraudulent or clear one.
